DIGITAL RESOURCE REQUEST SYSTEM

A web-based platform for requesting, managing, and approving digital learning resources. Developed a React.js frontend with an intuitive UI for users to submit, track, and manage digital resource requests. Built a Firebase-powered backend with Firestore, Authentication, Storage, and real-time notifications. Integrated an AI-powered recommendation system for personalized learning resource suggestions.

Key Features

  • 🔐 Secure Authentication (Firebase Auth)
  • 📄 Request & Approval Workflow
  • 🔔 Real-time Notifications
  • 🤖 AI-powered Resource Recommendations
  • 📂 File Upload & Storage

Tech Stack

  • ⚡ Frontend: HTML, CSS, JavaScript
  • 🔥 Backend: Firebase Firestore & Authentication
  • 🚀 AI: Machine Learning for Recommendations
  • ☁️ Hosting: Firebase Hosting
